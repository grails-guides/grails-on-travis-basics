### Test Logging

[source, groovy]
.build.gradle
----
tasks.withType(Test) {
    ...
    include::{sourceDir}/build.gradle[tag=testLogging]
    ...
}
----

### Run Tests

Run the tests with gradle to see that everything executes correctly up to this point.

`./gradlew check`.
